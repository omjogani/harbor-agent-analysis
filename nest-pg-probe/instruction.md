Use Ansible to install Docker. After installation, use Ansible to bring up a Docker Compose stack with two services - Postgres (latest) and a NestJS app. The NestJS app should be minimal (not a full scaffold) — just enough structure to expose a `GET /checkdb` endpoint that runs a `SELECT 1` query against Postgres and returns the result. The Compose setup should ensure NestJS only starts after Postgres is healthy. End with a `curl` to verify the endpoint works.
